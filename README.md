# KAG E.A.S.T University: Interactive Student IT Guide 🦁🎓

The official digital roadmap for students at **KAG E.A.S.T University**—the #1 Online University in Kenya. This interactive web application is designed to demystify the "Digital Hedge Maze" and empower students with the tools they need for independent learning and academic transformation.

## 🚀 Live Application
Access the guide here: **[https://kag-east-university.github.io/EAST-GUIDES/](https://kag-east-university.github.io/EAST-GUIDES/)**

---

## ✨ Features

### 🏛️ The Digital Front Door
A centralized hub for the three pillars of our digital campus:
*   **School Portal:** Administrative records, admissions, and fee structures.
*   **E-Learning (LMS):** The virtual classroom for unit materials and CATs.
*   **E-Library:** Research resources accessible via MyLOFT.

### 🎥 Video Training Vault
Embedded step-by-step video tutorials and walkthroughs covering:
*   Email activation and Portal unit registration.
*   LMS navigation and joining live BigBlueButton (BBB) classes.
*   Orientation "Master Guides" from the ICT department.

### 📚 Mastering the LMS (Deep-Dive)
Detailed, interactive instructions on:
*   **The Audio Trap:** Ensuring students join classes with the correct microphone permissions.
*   **The Submission Trap:** A critical guide on using the "Save Changes" button to avoid penalties.
*   **Assignment Workflow:** From downloading tasks to checking lecturer feedback.

### 🦏 Safe Exam Browser (SEB) Training
Essential "Digital Guardrails" for examinations:
*   Hardware requirements (Windows/macOS) and strict camera policies.
*   Step-by-step demo exam training.

### 🛟 ICT Support & Troubleshooting Clinic
*   Direct access to the **ICT Knowledge Base** and Ticketing System.
*   Contact details for the **IT Support Squad** (Paul, Eugene, Dancan, and Brian).
*   Information on the live **Troubleshooting Clinic** (May 15th).

## 📖 How to Update Materials

To update the downloadable survival guides:
1.  Navigate to the `/public/manuals/` directory.
2.  Replace the PDF files with the latest versions using the exact filenames specified in `App.tsx`:
    *   `Accessing LMS Materials.pdf`
    *   `Safe Exam Browser Training.pdf`
    *   `Complete LMS Student Manual.pdf`
3.  Commit and push the changes to GitHub.
